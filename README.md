# Liga Eterna: Los Guardianes del Juego

Web app educativa estática (HTML/CSS/JS) para practicar matemáticas con una temática de fútbol, pensada para niños de 6 y 8-9 años. Se publicará como PWA en GitHub Pages.

Consulta [`Docs/PLAN_TECNICO_Liga_Eterna.md`](Docs/PLAN_TECNICO_Liga_Eterna.md) para la arquitectura y el plan de trabajo completo.
